# FitGrid – Gym Management System

A desktop-based Gym Management System built using Python (Tkinter) for the GUI and SQL Server for the database backend.
This system is designed to streamline gym operations by providing structured management for both administrators and members.

# Features
1. Admin Panel

Add, update, and delete members

Manage membership plans and subscriptions

Track attendance records

View and manage stored database records

Secure login authentication

2. User Panel

View personal profile details

Check membership status

View attendance records

Access subscription information

# Tech Stack

Frontend: Python Tkinter

Backend: Microsoft SQL Server

Database Connectivity: pyodbc

Language: Python 

# Database Structure

The system uses a relational database model in SQL Server.
Main tables include:

Admins

Users

MembershipPlans

Attendance

Subscriptions

Relationships are structured using primary and foreign keys to ensure data integrity.

# Authentication System

The system includes role-based login:

Admin login

User login

Access to features is restricted based on role permissions.

# System Overview

The application provides:

Clean Tkinter-based graphical interface

Structured form handling

Secure data storage in SQL Server

Efficient CRUD operations

# Project Purpose

This project was developed to demonstrate:

GUI development using Tkinter

Database integration with SQL Server

Role-based access control

CRUD operations with structured backend connectivity
